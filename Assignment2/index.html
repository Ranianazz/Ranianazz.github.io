Video Art

Art video

This design was created to complement the art video. I chose to keep the video in its original location because the positioning worked well visually. To maintain a cohesive UI, I used a purple monotone that highlights the media player's colour and ties the entire website together. I implemented a pause button using a standard button element and connected it to a togglePlayPause function. For full-screen functionality, I used a full-screen icon and linked it to the updateFullscreen function. Initially, the icon was broken due to a rendering issue, so I used Boolean logic to prevent it from breaking again. I repositioned the control buttons to the top right corner and built a navigation bar for better accessibility. The buttons now include hover effects that change their colour and link to new pages. This interactive behaviour was achieved using HTML and CSS directly through source coding. The layout is responsive across different screen sizes, and I ensured basic accessibility by using semantic elements and making sure the buttons are keyboard-navigable and screen reader friendly.
